Blogs Panel

This panel displays created blog posts on the left to assist admin manage and track them. To the rigth is a form for creating a new blog post. Expanding a blog post displays more details on the blog and more management information about the same.

Running Blogs

These are blogs that are currently running and users are able to interact with them. You can expand any of them by clicking on them, through this, you can access a more comprehensive details and action buttons.
